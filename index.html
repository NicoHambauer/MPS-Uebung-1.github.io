Delivering Content with bootstrap

Delivering beautiful and readable content with bootstrap is simple. Here you can see an example of a section that shows text as well as an image in a row. To do this, simply use the element classes that bootstrap provides to generate a grid layout. Bootstrap uses a simple logic: every page is 12 sections wide. If you want to split up the width, use elements that sum up to 12 in width. For this section, we use two elements with a width of 6 (class="col-md-6"), as can be seen below: 


                <div class="row">
                  <div class="col-md-6">
                    Textcontent
                  </div>

                  <div class="col-md-6">
                    Image Content
                  </div>

                </div>

              </p>
              <input type ="text" id ="first"> + <input type ="text" id ="second">
              <p id="result">Result</p>
              <p><a class="btn btn-primary" role="button" id="calc">Calculate</a></p>
              </div>

The JavaScript Code


            <script>
              // Define an Event if button is clicked
              var calcbutton = document.getElementById("calc");
              calcbutton.addEventListener("click", calculate);
      
              // calculate function
              function calculate() {

                // Get the values of both inputs
                var first = parseInt(document.getElementById("first").value);
                var second = parseInt(document.getElementById("second").value);
      
                // Check if inouts are numbers, if so, output result, if not output warning
                if (isNaN(first) || isNaN(second)){
                  document.getElementById("result").innerHTML = "One of your inputs is not a number";
                } 
                else {
                  document.getElementById("result").innerHTML = first + second;
                }
                  
              }
              </script>
